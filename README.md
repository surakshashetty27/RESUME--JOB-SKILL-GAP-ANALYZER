# RESUME--JOB-SKILL-GAP-ANALYZER

**Project Overview**
A machine learning-based tool called the Resumeâ€“Job Skill Analyzer assesses how well a candidate's resume fits a particular job description by examining the necessary and possessed skills. The project extracts, cleans, and vectorizes skill-related content from job postings and resumes using natural language processing algorithms. The technology gives an objective measure of compatibility by calculating the resemblance between the two, which helps recruiters and job seekers make better decisions and expedite the hiring process.

**Dataset Used**
https://www.kaggle.com/datasets/meerawks/it-skills-from-jobs

**Key Features**
- Automated technical skill extraction from job descriptions and resumes
- Text preprocessing with NLP methods like stopword removal and tokenization
- Vectorization and similarity metrics, such as cosine similarity and TF-IDF, are used in skill matching.
- creates a match score that shows how well a resume fits the job requirements.
- Minimizes bias and manual labor in resume screening
- Scalable strategy that can manage several job roles and resumes
